# Orçamento-de-Aluguel
Este projeto foi desenvolvido como parte da disciplina Algoritmos e Programação Orientada a Objetos da UniFECAF.
O objetivo é aplicar conceitos de Python e POO para calcular orçamentos de aluguel de imóveis de forma automatizada.

O sistema recebe informações sobre o tipo de imóvel e suas características, aplica regras de negócio e gera um arquivo .csv com as parcelas mensais do contrato.
